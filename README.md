# README

jQuery日历插件，带有日期事件记录功能，可作为日程规划or日程记录。

jQuery版本：1.11.0

基于https://codecanyon.net/item/jalendar-2-calendar-pack-event-range-and-more/12662442，部分改动，细节调优。
